Contextualização

Surgimento da Mineração de Dados

A disseminação do uso de meios eletrônicos na sociedade moderna tem gerado uma enorme quantidade de dados.

  • Uso de SGBDs na maioria das organizações públicas e privadas de médio e grande porte.
  • Avanços na aquisição de dados, desde os leitores de códigos de barra até sistemas de sensoriamento remoto.

Cenário

Como analisar essa enorme quantidade de dados?

SQL e Mineração de Dados

  • Bancos de dados relacionais são responsáveis por armazenar e recuperar dados de forma eficiente.
  • Esses bancos de dados permitem a extração de diversas informações usando SQL.
  • Então para que serve a Mineração de Dados?
  • Usando SQL: esse processo resolve questões que necessariamente devem ser definidas.
  • Mineração de Dados: a descoberta de novas informações é automática, ou seja, sem que seja necessária a formulação de uma questão.

Mineração de Dados

O que é Mineração de Dados?

É o processo automático de descoberta de novas informações e conhecimento, úteis a uma aplicação, no formato de regras e padrões, “escondidos” em grandes volumes de dados.

Onde ocorre?

Este processo é executado sobre grandes quantidades de dados, estejam esses armazenados em bancos de dados tradicionais, em data warehouses ou em outras formas de repositórios.

Multidisciplinariedade

Tarefas e Técnicas em Mineração de Dados

Tarefas versus Técnicas

O processo de mineração de dados envolve o uso de diversas tarefas e técnicas.

  • Tarefas: são classes de problemas.
  • Técnicas: são os algoritmos utilizados na resolução dos problemas propostos nas tarefas.

Categorias de Tarefas

As tarefas de mineração de dados podem ser divididas nas seguintes categorias:

  • Tarefas Preditivas: do conhecimento adquirido a partir de um conjunto de dados, fazemos predições para novas instâncias.
  • Tarefas Descritivas: buscam identificar padrões de comportamento comuns nos dados.

Principais Tarefas de Mineração de Dados

Mineração de Dados Direta

Através de uma técnica de mineração, extrai-se ou treina-se um modelo que será posteriormente utilizado.

  • Principais tarefas: classificação e regressão.

Mineração caracterizada pela existência de um campo específico (atributo classe) cujo valor deve ser estimado a partir dos valores dos demais atributos (atributos preditores).

Mineração de Dados Indireta

Através de uma técnica de mineração, extraem-se padrões significativos que serão posteriormente avaliados.

  • Principais tarefas: extração de regras de associação, agrupamento.

O resultado da mineração complementa o conhecimento do especialista e deverá ser examinado e avaliado por este.

Extração de Regras de Associação

Objetivo

Identificação de itens de um mesmo domínio de aplicação que ocorrem juntos com determinada frequência na base de dados.

Exemplo: Market Basket Analysis \(\rightarrow\) identificação de produtos que são comprados juntos em um número significativo de transações de compras.

Regras de Associação Booleanas

Uma regra de associação representa um padrão de relacionamento entre itens de dados do domínio da aplicação que ocorre com uma determinada frequência na base de dados (transacional).

Id-Transação (TID) Itens Comprados
1 leite, pão, refrigerante
2 cerveja, carne
3 cerveja, fralda, leite, refrigerante
4 cerveja, fralda, leite, pão
5 fralda, leite, refrigerante
  • Exemplos de regras: {fralda} → {cerveja}, {fralda} → {leite}

Regras de Associação Quantitativas

São utilizadas quando se deseja minerar padrões em bases de dados relacionais (formadas por atributos quantitativos e atributos categóricos).

  • Exemplo:
    (Sexo = “M”) e (20 < Idade < 30) e (Profissão = “Advogado”) → (Compra = “SIM”)

    • Esta regra indica, com certo grau de certeza, que advogados, entre 20 e 30 anos, do sexo masculino são consumidores de laptops.

Extração de Padrões de Sequência

Definição

Padrões de sequência representam sequências de conjuntos de itens que ocorrem nas transações de diferentes consumidores, com determinada frequência (na ordem especificada).

  • Exemplo: {(leite) (queijo, manteiga)}

Agrupamento

Definição

Agrupamento é o processo de identificação de um conjunto finito de categorias (ou grupos - clusters), não previamente definidos, que contêm objetos similares.

  • Exemplo: Deseja-se separar os clientes em grupos de forma que aqueles que apresentam o mesmo comportamento de consumo fiquem no mesmo grupo.

Classificação e Regressão

Objetivo da Classificação

Identificar a classe a qual um elemento pertence a partir de suas características. O conjunto de possíveis classes é discreto e predefinido.

  • Exemplo: a partir das características de um indivíduo, determinar a que classe social ele pertence.
    • Conjunto de classes = {A, B, C, D, E}.

Categorias de tarefas de classificação: binária, multiclasse, multirrótulo e hierárquica.

Árvore de Decisão

Objetivo da Regressão

Estimar o valor de um atributo de uma instância a partir de suas características. O domínio deste atributo deve ser numérico e contínuo.

  • Exemplo: a partir das características de um imóvel, determinar seu valor de venda ou aluguel.

    \(Y=\alpha + a_1 x_1 + a_2 x_2 + \ldots + a_n x_n\).

Aplicações e Softwares

Aplicações

Analisar tendências e encontrar padrões a partir de dados históricos com o objetivo de prever ações futuras e apoiar decisões pode ser um procedimento útil em diversas áreas, tais como:

  • Marketing.
  • Finanças.
  • Saúde.
  • Educação.
  • Segurança.
  • Outras.

Softwares de Mineração de Dados

  • Weka: Desenvolvido em Java pela Universidade de Waikato, contém uma variedade de algoritmos de Data Mining (DM).
  • Rapid Miner: Desenvolvido pela empresa Rapid Miner, é uma ferramenta em Java que permite trabalhar com aprendizado profundo, mineração de texto e aprendizado de máquina.
  • Oracle Data Mining: Desenvolvido pela Oracle, permite interligação direta com o banco de dados Oracle.
  • Orange: Software desenvolvido em Python que fornece algoritmos de mineração de dados.
  • IBM SPSS Modeler: Software de mineração de dados utilizado para construção de modelos preditivos.
  • Scikit-learn: Software desenvolvido em Python que fornece um conjunto de algoritmos de mineração de dados.
